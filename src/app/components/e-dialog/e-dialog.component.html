Ejemplos básicos de cómo se visualiza y cómo utilizar e-dialog de forma personalizada.

Mostrar dialog <e-dialog title="Titulo" desc="Descripción" cancelText="Texto cancelar" confirmText="Texto confirmar"
   (confirmAction)="eventoClickConfirm()" [state]="dialogState"></e-dialog>

Propiedades

Documentación de propiedades que contiene el componente e-dialog.


title

Title que contendrá el dialog.

values ("" por defecto)

Tipo string

desc

Description que contendrá el dialog.

values ("" por defecto)

Tipo string

cancelText

Description que contendrá el button cancelar, si no hay descripción entonces no se mostrará el button de cancelar.

values ("" por defecto)

Tipo string

confirmText

Description que contendrá el button confirmar.

values ("Confirmar" por defecto)

Tipo string

confirmAction

Método que se ejecutará al clickar en el button confirmar.

values ("() => void" por defecto)

Tipo void

state

Estado del dialog.

values (false por defecto)

Tipo boolean
