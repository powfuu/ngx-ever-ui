Uso

Ejemplos básicos de cómo se visualiza y cómo utilizar e-dropdown de forma personalizada.

<e-dropdown titleDropdown="Lista de items" [clickAction]="eventoClickItem" [items]="items"></e-dropdown> items: string[] = ['Item 1', 'Item 2', 'Item 3'];

eventoClickItem(item: string): void {{ '{' }}
  alert(item + ' clicked!');
{{ '}' }}


Propiedades

Documentación de propiedades que contiene el componente e-dropdown.


titleDropdown

Title que contendrá el Dropdown.

values ("Dropdown" por defecto)

Tipo string

items

Lista de items que contendrá el Dropdown.

values ([] por defecto)

Tipo string[]

clickAction

Método que se ejecutará al clickar un item de la lista.

values ("() => void" por defecto)

Tipo void
