Propiedades

Documentación de propiedades que contiene el componente e-sidesheet.


state

Controla el estado para visualizar el sidesheet.

values (false por defecto)

Tipo boolean (true = abierto, false = cerrado).

size

Controla el tamaño del Sidesheet.

values ('m' por defecto)

s | m | xl

position

Controla la posición del Sidesheet.

values ('left' por defecto)

left | right
