Uso

Ejemplos básicos de cómo se visualiza y cómo utilizar e-button de forma personalizada.


Button Primary

Se utiliza para una acción principal.
Button <e-button>Button</e-button>

Button Secondary

Se utiliza para una acción secundaria.
Button <e-button theme="secondary" size="xxl">Button</e-button>

Button Third

Se utiliza para una acción que tiene un concepto tanto primario como secundario.
Button <e-button theme="third" size="s">Button</e-button>


Propiedades

Documentación de propiedades que contiene el componente e-button.


theme

Controla que tema de button utilizará.

values (primary por defecto)

primary | secondary | third

size

Controla el tamaño del button.

values (m por defecto)

s | m | xl
