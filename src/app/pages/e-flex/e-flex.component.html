Uso

Ejemplos básicos de cómo se visualiza y cómo utilizar e-flex content.


Flex Row

Se utiliza para que los elementos estén uno al lado de otro.

Row 1

Row 2

 <e-flex gap="m">
   <p>Row 1</p>
   <p>Row 2</p>
</e-flex>

Flex Column

Se utiliza para que los elementos estén uno debajo de otro.

Row 1

Row 2

 <e-flex fd="column" gap="s">
   <p>Row 1</p>
   <p>Row 2</p>
</e-flex>


Propiedades

Documentación de propiedades que contiene el componente e-flex.


fd

Controla que valor tendrá flex-direction.

values (row por defecto)

column | row

gap

Controla el espacio de separación entre cada elemento dentro del container.

values (n por defecto)

n | s | m | xl
