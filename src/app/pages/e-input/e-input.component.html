Uso

Ejemplo básico de cómo se visualiza y cómo utilizar e-input de forma personalizada.

<e-input placeholder="Buscar..." [(value)]="busquedaModelo"></e-input>


Propiedades

Documentación de propiedades que contiene el componente e-input.


placeholder

Controla que placeholder tendrá el input.

values ("" por defecto)

Tipo string | number, puede contener cualquier valor.

value

Controla el ngModel del input (siempre pasarle propiedad con [()] ya que el dato debe ser compartido entre componentes, ejemplo: value=[(value)])="nombreModelo".

values (null por defecto)

Puede contener cualquier modelo tipo string.
