Uso

Ejemplos básicos de cómo se visualiza y cómo utilizar e-loader de forma personalizada.

Mostrar loader por 3 segundos <e-button (click)="initLoader()">Mostrar loader por 3 segundos</e-button>
<e-loader [show]="loadingState" text="Cargando por 3 segundos..."></e-loader>


Propiedades

Documentación de propiedades que contiene el componente e-loader.


text

Controla que texto tendrá el loader.

values ("Loading..." por defecto)

Tipo string, puede contener cualquier valor.

show

Controla el estado del loader.

values (false por defecto)

Tipo boolean, false = cerrado, true = abierto.
