Uso

Ejemplo básico de cómo se visualiza y cómo utilizar e-search de forma personalizada.

<e-search placeholder="Buscar..." [(value)]="busquedaModelo"></e-search>


Propiedades

Documentación de propiedades que contiene el componente e-search.


placeholder

Controla que placeholder tendrá el search input.

values ("" por defecto)

Tipo string, puede contener cualquier valor.

value

Controla el ngModel del input (siempre pasarle propiedad con [()] ya que el dato debe ser compartido entre componentes, ejemplo: value=[(value)])="nombreModelo".

values (null por defecto)

Puede contener cualquier modelo tipo string.
